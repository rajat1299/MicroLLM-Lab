# Demo Script

1. Open app and choose `Regex Patterns` pack.
2. Set config: `n_embd=32`, `n_head=4`, `n_layer=1`, `steps=120`.
3. Click **Start Run**.
4. Show live updates in:
   - Loss Curve
   - Token Timeline
   - Attention Heatmap
   - Gradient and Update Norms
5. Open operation graph panel and use `Play` then `Step`.
6. Upload a `.txt` corpus and run again using `upload:<id>` pack.
7. Trigger cancel on a running job to show cancellation path.
